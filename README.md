# MyOpenClaw - Todo List

一个基于 Next.js + TypeScript + Tailwind 的现代化 Todo List。

## 功能
- 添加/完成/删除待办
- 过滤（全部 / 进行中 / 已完成）
- 清除已完成
- 本地存储（LocalStorage）
- 实时黄金价格（GoldAPI）

## 运行
1) 配置环境变量（推荐）：
```bash
cp .env.example .env.local
# 填入 GOLDAPI_TOKEN
```

2) 启动：
```bash
npm install
npm run dev
```

浏览器访问：http://localhost:3000
